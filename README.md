# BetterTouchpad

简洁、可配置的小米平板触控板增强工具。将触控板映射成虚拟鼠标，以在游戏/串流场景中拥有更好的兼容
提供边缘内划、双指滚动、双指轻触拖移等手势，并支持轴交换与反转、自然滚动等设置。

## 主要特性
1. 边缘内划（Edge Swipe）
   1. 注入的触摸起点固定在屏幕左右边缘
   2. 在开启 XY 轴对调时，起点仍位于左右边缘
   3. 注入触控点仅在水平方向移动（垂直方向可居中或按设置）
2. 双指滚动（Two-finger Scroll）
3. 双指轻触以拖移（Two-finger Tap & Drag）
   1. 支持可配置的双击间隔（毫秒）
4. 轴设置
   1. XY 轴对调（Swap X/Y）
   2. X 轴反转（Invert X）
5. 滚轮设置
   1. 自然滚动（Natural Scrolling）开关
6. 支持 NDK/CMake 原生模块（arm64-v8a、x86_64）

## 配置说明
1. 设置页面包含以下开关与选项：
   1. `边缘内划`：开启/关闭
   2. `XY 轴对调`：打开后触控逻辑以对调坐标处理输入输出
   3. `X 轴反转`：水平反向
   4. `自然滚动`：启用/禁用滚动方向的倒置
   5. `双指轻触以拖移`：开启后可设置 `双击间隔（ms）`
2. 边缘内划实现要点：
   1. 在进入边缘内划时固定注入起点为屏幕左右边缘坐标
   2. 后续仅根据触控板上手指水平位移累加注入位置，保证注入点只在水平方向移动
